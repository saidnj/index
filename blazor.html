BLAZOR

es un framework de desarrollo web de código abierto, creado por Microsoft, que permite construir aplicaciones interactivas en la web usando C# en lugar de JavaScript. Blazor se basa en .NET y permite a los desarrolladores utilizar sus conocimientos en C# y .NET para crear aplicaciones web modernas, sin necesidad de escribir código en JavaScript o depender de frameworks como Angular o React.
